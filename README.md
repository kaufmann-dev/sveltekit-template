# Project Templates

A collection of project templates pre-configured for OpenCode and Codex AI development.

To scaffold any of these templates into a new directory, run the corresponding `degit` command.

## Astro

Pre-configured Astro template with the Astro Docs MCP server.

```bash
npx degit kaufmann-dev/project-templates/astro .
```

## SvelteKit

Pre-configured SvelteKit template with SvelteKit OpenCode plugin integration.

```bash
npx degit kaufmann-dev/project-templates/sveltekit .
```

## Next.js

Minimal Next.js template folder structure.

```bash
npx degit kaufmann-dev/project-templates/next-js .
```

## Configuration

Each template includes standard workspace configurations:
- **OpenCode**: `.config/opencode/opencode.jsonc`
- **Codex**: `.codex/config.toml`
